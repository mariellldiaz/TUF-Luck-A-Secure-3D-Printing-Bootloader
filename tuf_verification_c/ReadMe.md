## Instructions

**Ubuntu** for Monocypher:
- Clone Monocypher github
git clone https://github.com/LoupVaillant/Monocypher.git

1. cargo build
2. cargo run --bin hash_firmware

**Ubuntu** for cJSON:
- Clone cJSON github
 git clone https://github.com/DaveGamble/cJSON.git

1. cd build
2. cmake ..
3. make
